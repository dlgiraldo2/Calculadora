# 🧮 Calculadora Android

Aplicación de calculadora para Android desarrollada como proyecto de estudio en el **SENA** (Servicio Nacional de Aprendizaje).

---

## 📋 Descripción

Aplicación móvil de calculadora básica desarrollada con Kotlin para Android. Permite realizar operaciones matemáticas fundamentales con una interfaz limpia e intuitiva. Desarrollada en Android Studio como parte del proceso de formación en desarrollo móvil del SENA.

## ⚙️ Funcionalidades

- ➕ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División
- 🔄 Limpiar pantalla
- 🔢 Soporte de decimales

---

## 📱 Requisitos

- Android 5.0 (API 21) o superior
- Android Studio para compilar

---

## 🚀 ¿Cómo ejecutar el proyecto?

1. Clona el repositorio:
```bash
git clone https://github.com/dlgiraldo2/Calculadora.git
```
2. Abre el proyecto en **Android Studio**
3. Espera que Gradle sincronice las dependencias
4. Ejecuta en un emulador o dispositivo físico con **Run ▶**
